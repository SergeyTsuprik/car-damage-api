Частые вопросы

Какая точность модели? ▼

mAP 0.87 на test датасете из 5,000 фото. Обнаруживает: царапины, вмятины, трещины, разбитое стекло, вмятины на боках.

Как быстро ответ API? ▼

Средний ответ: 1.2 секунды. Максимум: 3 секунды (для больших фото). Зависит от размера и качества изображения.

Какие форматы фото поддерживаются? ▼

JPG, PNG, WebP. Размер до 10MB. Рекомендуемое разрешение: 720x480 и выше.

Есть ли обучение модели на моих данных? ▼

Да, в плане Enterprise. Свяжись с нами: support@yourdomain.by

Где хранятся мои фото? ▼

Фото удаляются сразу после анализа. Не сохраняются на серверах. Полная конфиденциальность.

Примеры использования

🐍 Python

import requests

API_URL = "https://car-damage-api.railway.app/api/detect"
API_KEY = "test-free-key"

with open("car.jpg", "rb") as f:
    files = {"file": f}
    headers = {"X-API-Key": API_KEY}
    response = requests.post(API_URL, files=files, headers=headers)

data = response.json()
print(f"Найдено повреждений: {data['count']}")

💻 JavaScript

const form = new FormData();
form.append("file", fileInput.files[0]);

fetch("https://car-damage-api.railway.app/api/detect", {
  method: "POST",
  headers: {"X-API-Key": "test-free-key"},
  body: form

.then(r => r.json())
.then(data => console.log(data));

🔧 cURL

curl -X POST "https://car-damage-api.railway.app/api/detect" \
  -H "X-API-Key: test-free-key" \
  -F "file=@car.jpg"
