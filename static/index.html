API Примеры

cURL

curl -X POST https://car-damage-api-production.up.railway.app/api/detect \
  -H "X-API-Key: sk_YOUR_API_KEY" \
  -F "file=@/path/to/image.jpg"

Python

import requests

response = requests.post(
    "https://car-damage-api-production.up.railway.app/api/detect",
    headers={"X-API-Key": "sk_YOUR_API_KEY"},
    files={"file": open("image.jpg", "rb")}
)

print(response.json())

JavaScript

const formData = new FormData();
formData.append("file", fileInput.files[0]);

const response = await fetch(
    "https://car-damage-api-production.up.railway.app/api/detect",
    {
        method: "POST",
        headers: {"X-API-Key": "sk_YOUR_API_KEY"},
        body: formData
    }
);

const data = await response.json();
console.log(data);

Часто задаваемые вопросы

Как получить API key?

После регистрации API key будет отправлен на ваш email. Его можно использовать для всех API запросов.

Какие форматы изображений поддерживаются?

PNG, JPG, WebP. Максимальный размер файла 10MB.

Как изменить тариф?

Вы можете изменить план в любое время. Изменение вступит в силу со следующего дня.

Есть ли гарантия возврата денег?

Да, 30 дней на любой платный план. Свяжитесь с поддержкой support@kastikcars.com
