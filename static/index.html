Частые вопросы

Как быстро ответ API? ▼
Средний ответ: 1.2 секунды. Максимум: 3 секунды (для больших фото). Зависит от размера и качества изображения.
Какие форматы фото поддерживаются? ▼
JPG, PNG, WebP. Размер до 10MB. Рекомендуемое разрешение: 720x480 и выше.
Где хранятся мои фото? ▼
Фото удаляются сразу после анализа. Не сохраняются на серверах. Полная конфиденциальность.

Примеры использования

🐍 Python (Web UI - Session Token)

import requests API_URL = "https://api.example.com/api/detect" session_token = "YOUR_SESSION_TOKEN" with open("car.jpg", "rb") as f: files = {"file": f} headers = {"X-Session-Token": session_token} response = requests.post(API_URL, files=files, headers=headers) data = response.json() print(f"Найдено повреждений: {data['count']}")

🔑 Python (API Key)

import requests API_URL = "https://api.example.com/api/detect" api_key = "sk_YOUR_API_KEY" with open("car.jpg", "rb") as f: files = {"file": f} headers = {"X-API-Key": api_key} response = requests.post(API_URL, files=files, headers=headers) data = response.json() print(f"Стоимость: ${data['cost']}")

💻 JavaScript (API Key)

const form = new FormData(); form.append("file", fileInput.files[0]); fetch("https://api.example.com/api/detect", { method: "POST", headers: {"X-API-Key": "sk_YOUR_API_KEY"}, body: form }) .then(r => r.json()) .then(data => console.log(data));

🔧 cURL (API Key)

curl -X POST "https://api.example.com/api/detect" \ -H "X-API-Key: sk_YOUR_API_KEY" \ -F "file=@car.jpg"
