Примеры кода

cURL

curl -X POST \
  https://car-damage-api.com/api/detect \
  -H "X-API-Key: YOUR_API_KEY" \
  -F "file=@photo.jpg"

Python

import requests

with open('photo.jpg', 'rb') as f:
    r = requests.post(
        'https://car-damage-api.com/api/detect',
        headers={'X-API-Key': 'YOUR_API_KEY'},
        files={'file': f}
    )
    print(r.json())

JavaScript

const formData = new FormData();
formData.append('file', fileInput.files[0]);

fetch('https://car-damage-api.com/api/detect', {
    method: 'POST',
    headers: {'X-API-Key': 'YOUR_API_KEY'},
    body: formData
})
.then(r => r.json())
.then(data => console.log(data))

Частые вопросы

Какая точность модели?

mAP 0.87 на test датасете. Обнаруживает: царапины, вмятины, трещины, разбитое стекло.

Как быстро ответ API?

Средний ответ: 1.2 секунды.

Какие форматы фото поддерживаются?

JPG, PNG, WebP. Размер до 10MB.

Безопасность данных?

Все загруженные фото удаляются сразу после анализа. Данные не хранятся.
